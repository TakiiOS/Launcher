**Cloner et installer les dépendances**

```console
> git clone https://github.com/TakiiOS/Launcher.git
> cd Launcher
> npm install
```
---

**Lancer l'application**

```console
> npm start
```
---

**Création du build**

| Platform    | Command              |
| ----------- | -------------------- |
| Windows  | `npm run build:win`   |
| macOS    | `npm run build:mac`   |
| Linux    | `npm run build:linux` |
